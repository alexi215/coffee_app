# Buzz Finder

### Overview
**Buzz Finder** a simple app that gives users the ability to search for nearby coffee shops.
* [Link to deployed site on heroku](http://arcane-badlands-7706.herokuapp.com/)

### Technologies Used

* Ruby 1.9.3p32
* JavaScript
* SCSS
* Ruby on rails 4.1.6
* Google Places Library
* Heroku
